# Leamout

Leamout is a paused prototype for mobile-money-first billing and monetization infrastructure for African digital creators.

The project is not under active development at the moment due to **Compliance and Regulatory Burden**.

## Status

**Paused.**

Leamout explored a backend foundation for creator subscriptions, renewal reminders, checkout sessions, communication credits, and mobile-money-oriented payment flows. The prototype was useful for learning, architecture exploration, and validating product direction, but it is not ready for production use.

## Why the project is paused

Billing and payment infrastructure is not only a technical product. It carries serious legal, financial, operational, and compliance responsibilities, including areas such as:

- payment-provider obligations;
- merchant onboarding and verification;
- consumer protection;
- fraud and abuse handling;
- transaction monitoring;
- refunds, disputes, and reconciliation;
- data privacy and security;
- tax, reporting, and regulatory expectations across markets.

For a solo or early-stage builder, these requirements create a level of operational weight that should not be ignored. Leamout is therefore paused until there is a clearer path for handling the compliance and regulatory responsibilities properly.

## What this repository contains

This repository may contain experimental backend work around:

- products and prices;
- customers and subscriptions;
- renewal and dunning flows;
- checkout sessions;
- personal access tokens for API testing;
- communication credits for SMS reminders;
- mobile-money payment provider experimentation;
- local development tooling.

The code should be treated as prototype work, not production-grade financial infrastructure.

## Production readiness

Leamout is **not production ready**.

Do not use this repository to process real customer payments, manage live subscriptions, hold customer funds, operate as a payment intermediary, or provide regulated financial services.

## Future

There is no active public roadmap for Leamout at this time.

Development may resume only if the compliance, regulatory, security, and operational responsibilities can be handled responsibly.
